Markdown

– это облегченный язык разметки, который является инструментом преобразования кода в HTML.

Заголовки

первого, третьего и шестого уровней, выполненные с помощью символа («#»), выглядят следующим образом:

    # text

    ## text
    ### text

Линия

  • Поставьте -- и появиться линия разделения 'hr'

Выделение текста

  • Текст, окружённый «**», будет заключен в HTML-тэг 'strong'.
  • Текст, окружённый «~~», будет заключен в HTML-тэг 'strike'.

Выделение текста цветом

  • Текст, окружённый «[++text++]», будет выделен зеленым цветом.
  • Текст, окружённый «[--text--]», будет выделен красным цветом.
